HR Payroll Multi-Currency Journal

You can easily create payslip with multi currency

Odoo HR Payroll Multi-Currency for create payslip using different currency.

Notes: this module support enterprise edition only

This Extension Allows

  • Apply currency based on Journal assigned in Salary Structure.
  • Affect correctly to accounting based on accounting date.

Journal View

Determine currency for journal

Salary Structure View

Choose Journal for each salary structure.
Do not forget to choose it default to automatically set when generate payslip.

Contract View

Choose Structure Type, system will apply currency based on default salary structure

Payslip View

When generate payslip, system will appear currency based on journal defined in salary structure if no currency defined, system will get company currency
Print payslip with currency

Journal Entry

Confirm payslip to generate journal entry

Payslip Analysis Report

Allow to group by and filter by currency (New in version 14.0.1.1.0)
